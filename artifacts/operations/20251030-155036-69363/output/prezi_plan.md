# Prezi Plan: Applications 2022 vs 2025 — Emergence in the Age of Singularity

Goal
- Tell a coherent, zoomable story: pre- vs post-LLM job application behavior (2022 vs 2025), framed by the “Emergence in the age of singularity” theme.

Structure (Zoom Path)
1) Overview (Start)
   - Big idea: “From burst to sustained velocity: post-LLM cadence.”
   - Visual: Hero panel with title and motif (see Emergence theme below).
2) Data Core
   - Node A: Yearly volume bar → applications_by_year_bar.png
   - Node B: Monthly timeline → applications_by_month_line.png
   - Node C: Year×Month heatmap → year_month_heatmap.png
3) Focus: 2022 vs 2025
   - Node D: Monthly overlay → compare_2022_vs_2025_monthly.png
   - Node E: Cumulative overlay → compare_2022_vs_2025_cumulative.png
   - Node F: Weekday comparison → compare_2022_vs_2025_weekday.png
4) Company Lens
   - Node G: Top Companies 2022 vs 2025 (two frames or split view)
5) Bursts & Capacity
   - Node H: Weekly timeline (bursts) → applications_by_week_line.png
   - Node I: Capacity heuristics (10-day, 4-week, 16–18 weeks)
6) Next Layer (Inbox/Funnel)
   - Node J: Gmail + ATS integration plan: response, interview rates
7) Closing (End)
   - Takeaways: Post-LLM persistence, higher throughput, next metrics.

Design System
- Colors: charcoal bg (#2C2C2C), off-white text (#F4F6F6), 2022 blue (#4C78A8), 2025 orange (#F58518), silver accents (#AAB7B8)
- Fonts: Arial throughout; bold for headers, 16–18pt body
- Motion: Zoom to reveal comparisons, pan between focus clusters (2→3→4→5→6)

Prezi Build Notes
- Template: choose a dark, modern template with zoomable clusters (Overview → Data Core → Focus → Company → Bursts → Next → Closing)
- Place each node as a circle/hexagon; use lines to indicate sequence
- Use subtle zoom-in for overlays; avoid excessive spins
- Keep caption text short; data labels handled in images
