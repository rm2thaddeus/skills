Le rendu attendu est un fichier Excel construit à partir de ce fichier test avec les indicateurs demandés correctement renseignés dans la feuille "dashboard"
Le fichier test nous a été transmis par le chef de produit de "Produimax" du laboratoire Umbrella Corp et il souhaite analyser les données de visites de ses délégués médicaux.
Le chef de produit s'intéresse uniquement aux visites effectuées sur l'année 2020 dans les régions Nord (S4,S5, S6) et Sud (S1, S2, S3).
Le fichier test est construit de la façon suivante :
Données médecins Contient les données brutes concernant les médecins, leurs secteurs, leurs spécialités,
Données visites Contient les données des visites effectuées auprès des médecins par les délégués médicaux, les produits mentionnés, le mode de visite
Dashboard Contient les différents indicateurs à créer
La note du test est construite comme suit : 60% sur les indicateurs correctement calculés
30% sur les étapes de construction intermédiaire des indicateurs
10% sur la propreté et la lisibilité du fichier rendu
Le fichier doit être envoyé 2h après son envoi par mail :  tout fichier reçu après cela ne sera pas corrigé
